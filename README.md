# Godette

A small Zelda-inspired action adventure built in **Godot 4.6** using a low poly art style with assets from **KayKit**.

This project is a small personal indie game experiment created as a special birthday surprise for my son.

The goal is not to build a fully polished commercial game, but to create a charming, playable adventure with memorable moments, fun exploration, and simple but satisfying combat.

The focus is on finishing something small and meaningful rather than building a huge endless project.

---

## Project Vision

**Inspired by:**

* The Legend of Zelda
* Tunic
* Death’s Door
* Hyper Light Drifter

**Core Focus:**

* Responsive melee combat
* Simple ranged combat (bow + arrows)
* Exploration with meaningful rewards
* Small handcrafted world
* Dungeon progression with keys and locked areas
* Low poly stylized presentation

This is intentionally a **small-scope project**.

It is not meant to be a full RPG or a large open world game.

The goal is a fun little adventure that feels special.

---
